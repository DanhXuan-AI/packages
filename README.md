# Package Repository

## Overview

Welcome to the DX Advisory governed package mirror. This repository is maintained by DX Advisory to store and preserve package availability for internal and approved client-system support. These packages support controlled DX Advisory workflows and approved client-reference systems.

## Purpose

The primary objective of this repository is to:

- Maintain a backup of key packages used within DX Advisory internal and approved client-support systems.
- Ensure the availability and accessibility of these packages at all times.

## Package List

| Package   | Description                                                                                                              | 
|-----------|--------------------------------------------------------------------------------------------------------------------------|
| chromium  | Chromium is an open-source web browser project, serving as the foundation for Google Chrome.                             |
| wkhtmltox | Wkhtmltox is a command line tool to render HTML into PDF and various image formats using the Qt WebKit rendering engine. |
| yt-dlp    | YouTube downloader videos.                                                                                               |
